💼 Sales Process

Step-by-step guide for all three sales scenarios

For EXISTING customers wanting AMC or upgrade — skip to AMC Sale or Product Upgrade tabs. Do NOT create a Lead.
1Create Lead

Capture: Full Name, Mobile, Email, Enquiry Source, Product of interest. Only for new prospects who have never been a customer.

Lead DocTypeSource: Walk-in / Referral / Online / Event
2Qualify → Create Opportunity

Check: Need, Budget, Decision-maker. If qualified → Create Opportunity from the Lead. Set Product, Value, Probability, and initial Stage.

Opportunity must come from a Lead for new customers.
3Follow-up Pipeline

Move through pipeline stages: New → Contacted → Demo Done → Proposal Sent → Negotiation → Won/Lost. Log every activity.

4Create Quotation

Always create from the Opportunity — never from Lead directly. Include product, quantity, price, terms.

Quotation must be linked to an Opportunity, NOT a Lead.
5Convert to Customer

On acceptance, convert Quotation → Customer DocType is created. This is the master record for the business relationship.

6Sales Order + Proforma Invoice

Create Sales Order. Generate Proforma Invoice and share with customer for payment reference.

7Payment + Sales Invoice

Record Payment Entry when payment received. Then submit Sales Invoice as the official tax invoice.

8Create Customer Product + AMC (if applicable)

If product is AMC-eligible: Create Customer Product record, then create AMC. Set Type (Paid / Complementary / Deferred), Start Date, Expiry = Start + 365 days.

Existing customer — do NOT create a Lead. Start from Customer record directly.
1Create Opportunity from Customer

Type: AMC Sale. Navigate to Customer record → Create Opportunity. This ensures the opportunity is correctly linked to the existing relationship.

2Follow-up Pipeline

Work through stages, log activities. If price negotiation needed → Create Quotation. If straightforward → proceed directly to Invoice.

3Invoice + Payment

Raise Sales Invoice. Record Payment Entry when payment is received.

4Create AMC

Select AMC Type. For Deferred: Start = Invoice date + 1 year. For Paid / Complementary: Start = Invoice date + 1 day. Expiry = Start + 365 days.

Existing customer — do NOT create a Lead. Start from Customer record.
1Create Opportunity from Customer

Type: Product Renewal / Upgrade. Link to existing Customer record. Specify the product being upgraded and the new version/tier.

2Quotation (if needed) → Invoice

If pricing is straightforward → create Invoice directly. If negotiation needed → create Quotation first, then Invoice on acceptance.

3Update Customer Product

Update the Customer Product record with the new version, tier, and expiry. This keeps the customer's product history accurate.

🔄 Renewal Scenarios

✅ Easy Renewal

Customer responds at same price → Direct Invoice from Customer → Skip Opportunity → Payment Entry → AMC renewed

📞 Follow-up Renewal

No response to reminders → Create Opportunity from Customer → Follow-up pipeline → Quotation (if negotiation) → Invoice → AMC renewed

💀 Lapsed AMC

No renewal after 90+ days → AMC marked Lapsed → Manual re-activation required → Re-activation fee may apply

💡 Common Mistakes to Avoid

❌ Creating Leads for existing customersAlways check if the person is already a customer before creating a Lead.
❌ Creating Quotation directly from LeadAlways create Opportunity first, then Quotation from Opportunity.
❌ Processing tickets for expired AMCAlways validate AMC status before accepting any ticket.
❌ Wrong AMC start datesUse the correct formula per AMC type — not the current date or invoice date for renewals.
❌ Forgetting billable check on ticket closureEvery closed ticket needs a billable review. Missed invoices = lost revenue.
